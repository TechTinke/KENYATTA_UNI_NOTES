Boolean Operators 
1. && - AND     e.g exp1 && exp2
- TRUE(1) only if both exp1 and exp2 are true otherwise FALSE(0)
Examples
(5==5)&&(6!=2) - TRUE(1) because both operands are true 
(2==1)&&(5==5) - FALSE(0) because one operand is false 

2. || - OR      e.g exp1 || exp2
-TRUE(1) if either exp1 or exp2 is true; FALSE(0) if only both are false
Example

3. ! - NOT      e.g !exp1
- FALSE(0) if exp1 is true; TRUE(1) if exp1 is false
Example
