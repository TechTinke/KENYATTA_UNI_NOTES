THE COMPONENTS OF A C PROGRAM
- Every c program consists of several components combined in a certain way.
a. main() Function
-Only component that is required in ecery C program is the main() function
-Consists of the name main followed bby a pair of empty parantheses() and a pair of braces{}
-Under nnormal circumstances, program execution starts at the first statement in main() and terminates at the last statement in main()

b. Braces 
- You use braces {} to enclose the program lines that make up evry C function - including the main() function 
- A group of one or more statements enclosed within braces is called a block 

c. The #include Directive
- instructs the C compiler to add the contents of an include file into your program during compilation
- contains information needed by your program or the compiler
- sometimes called header files
- all have an .H extension (for example, STUDIO.H)
- You use the #iclude directive to innstruct the compiler to add a specific include file to your program during compilation

d. The Variable definition
- A variable is a name assigned to a data storage location 
- Program uuses variables to store various kinds of data during program execution 
- In C, a variable must be defined before it can be used.
- A variable definiton informs the compiler off the variable's name and the type oof data it is to hold

TYPES OF Variables
-Integers
-Float 
-Boolean
-Characters
-Strings

e. The Function Prototype
- A FUNCTION PROTOTYPE provides the C compiler with the name anf arguments of the functions contained in the program 

f. Program statements
- Do the real work of a C program
- Display infromation on-screen, read keyboard input, perform mathematical operations, call functions, read disk files, and carry out all the other operations that a program needs to perform
- Always end with a semicolon;
Examples:
a. printf()
b. scanf()
c. return

g. The Function Definition 
- A Function is an independent, self-contained section of code that is written to perform a certain task.
- Every function has a name, and the code in each function is executed by including that function's name in a program statement. This is known as calling the function

h. Program comments
- Any part of your program that starts with / and ends with / or has // is called a comments
- The compiler ignores all comments as they have no meaning to te source code
- A comment can span part of a line, an entire line or multiple lines 
- Here are three examples:
a. /a single line comment/
b. use // for a single line comment only 
  
  USES OF COMMENTS 
- Documentation of variables and their usage

- Describing the author, program, date, modification changes, revisions
